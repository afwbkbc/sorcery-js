List of publicly available Sorcery methods:

Sorcery.initialize(callback)
	must be called prior to any use of Sorcery, usually right after including sorcery.js
	all Sorcery-related code must reside in callback

Sorcery.require(dependables,callback)
	similar to RequireJS's require()
Sorcery.define(dependable,callback)
	similar to RequireJS's define()

Sorcery.get_require_paths()
	returns an array containing paths that Sorcery.require and Sorcery.define search requireables in
