provides console-related methods

Cli.print(text)
	outputs text to console, without training \n . add \n manually to end lines.

Cli.get_arguments()
	returns array of raw command-line arguments (e.g. [ '--debug=false', '--test=55' ] )
	
Cli.get_parameters()
	returns array of parsed key-value parameters found (e.g. { debug: false, test: 55 } )
	parsing similar to readline
		--no-debug , --without-debug , --disable-debug , --debug=false , --debug=no are all the same
		--debug , --with-debug , --enable-debug , --debug=true , --debug=yes are all the same
	parameters without leading -- are added as numeric indexes starting from 0
		so "node console.js testapp --with-debug" will be parsed as { '0': 'testapp', debug: true } and testapp will also be accessible by Cli.get_parameter(0)
		  
Cli.get_parameter(key)
	returns specific parameter or null

Cli.get_node_path()
	returns path of node used to run application (e.g. "node" or "/usr/bin/node")
	
Cli.get_app_path()
	returns path of application (e.g. "console.js" or "/var/www/sorceryapp/console.js")
