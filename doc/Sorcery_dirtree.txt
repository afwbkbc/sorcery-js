README.md

sorcery.js
	main file to be included or required, both in cli and web
	
console.js
	used for various cli tasks
	run as node console.js <task> [<parameters>]



following directories are used for searching of requireables (e.g. library/cli), in specified order:
	app/
		your local application directory
	packages/*
		directories with external packages (e.g. packages/sorcery)
		you can add additional packages to search path with Sorcery.add_package, only packages/sorcery is available by default

* in case of name collision, you can include requireable from specific package by specifying full path (sorcery/library/cli or even packages/sorcery/library/cli)
